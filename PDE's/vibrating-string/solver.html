PDE Solver - Vibrating String

This app solves the partial differential equation for a vibrating string:






using an explicit, numerical, algorithm. It discretizes the equations and then solves the equation for each position and timestep, using the y-position from the previous time steps. To see how the string's position changes over time, drag the slider. If you would like to change the algorithm parameters, input the parameters that will change into the input boxes below and hit the "Redo!" button. The defaults are h = 0.05 (x-width), k = 0.01 (t-width), T = 2 (max time to solve until).
